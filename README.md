### Step by step Gradle project

Int Gralde project
```
gradle init 

```

Clean, Build, Run (without unitTest)
```
./gradlew clean
./gradlew build
./gradlew run

gradle clean build run 
```

Check all task
```
./gradlew tasks
```

without unitTest
```
./gradlew build -x test
./gradlew run -x test
```

Ref: https://hackersandslackers.com/building-java-projects-with-gradle/
